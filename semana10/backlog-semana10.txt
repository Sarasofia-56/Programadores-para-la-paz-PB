Backlog priorizado

1. Como ciudadano quiero registrar una incidencia para dejar evidencia del caso.

2. Como coordinador quiero recibir una alerta en Telegram cuando se registre un nuevo reporte.

3. Como miembro del equipo quiero consultar los reportes para hacer seguimiento a los casos.

4. Como administrador quiero actualizar el estado de un reporte para informar el avance.

5. Como ciudadano quiero recibir una notificación cuando mi reporte haya sido atendido.
