El login permite identificar a un usuario antes de darle acceso a determinadas funciones de la plataforma. Esto ayuda a proteger la información y controlar quién puede realizar acciones importantes. Algunas rutas deben estar protegidas porque contienen datos privados o permiten modificar información del sistema. No deberían exponerse públicamente contraseñas, documentos de identidad, direcciones, teléfonos, tokens, claves API ni otros datos sensibles. La autenticación fortalece la seguridad y genera confianza en los usuarios.
