Al enviar una petición POST a la ruta /registro con el JSON:

{
  "nombre": "Maria",
  "mensaje": "Hola comunidad"
}

el servidor recibe los datos mediante req.body y responde con:

{
  "estado": "Datos recibidos",
  "nombre": "Maria",
  "mensaje": "Hola comunidad"
}

Esto confirma que la información fue recibida y procesada correctamente.
